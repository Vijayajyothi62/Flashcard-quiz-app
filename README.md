# Flashcard Quiz App

A simple Flashcard Quiz App built using HTML, CSS, and JavaScript.

## Features

- Display questions on flashcards
- Show answer with one click
- Navigate using Previous and Next buttons
- Add new flashcards
- Edit existing flashcards
- Delete flashcards
- Simple and clean user interface

## Technologies Used

- HTML
- CSS
- JavaScript
